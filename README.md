### Wiring Information

| L293D  | Node MCU |
| ------------- | ------------- |
| IN1  | D1  |
| IN2  | D2  |
| IN3  | D3  |
| IN4  | D5  |
| ENA  | D6  |
| ENB  | D7  |
| GND  | GND  |

| Source  | Pins |
